Geoscience AI Software Landscape

Disrupting technologies, products, and interactive UI flows across the geological exploration pipeline

🏗️
Foundation models (LLMs for geoscience)

Domain-specific large language models trained on geological corpora — enabling natural language Q&A, document mining, and multimodal map analysis for exploration teams.

GeoGPT
Multimodal geological Q&A — upload maps, query in natural language
Research / early access
UI Flow
Upload geological map Type natural language query AI extracts lat/lon + legend Interactive dialogue Q&A Export findings
Gap: cannot reason over 3D geometry
RadiXplore
Mining-specific archive intelligence — search millions of scanned geological reports
Commercially deployed
UI Flow
Connect historical archive Type natural language query View ranked results + evidence highlights Clip anomalies / targets Push to target generation workflow
🌊
Generative / diffusion models

Probabilistic subsurface simulation from sparse drill data. 3D latent diffusion models enable facies-based geological realizations with variable metaparameters — far beyond 2D fixed-parameter systems.

RMSP — GeologicAI
Conditional simulation engine for resource model generation
Conditional sim deployed; true diffusion in development
UI Flow
Import sparse drill data Configure simulation parameters Generate probabilistic realisations Review uncertainty per block Export block model
Seequent Evo
Cloud geological modelling platform with probabilistic subsurface realisation
Conditional sim; diffusion R&D
UI Flow
Load drillhole / geophysics data Define geological domains Run conditional simulation Visualise realisations in 3D History-match against production data
Gap: true diffusion-based 3D generation not yet productized
🧠
Physics-informed neural networks (PINNs)

Neural networks trained with physical constraints (petrophysical laws, geostatistical rules) embedded directly into their architecture — ensuring predictions respect geological physics even with limited data.

Maptek Vestrex / DomainMCF
Physically-constrained RBF interpolants for complex structural geology
Commercially deployed
UI Flow
Import drillhole + structural data Select RBF interpolant type (Linear / Spheroidal) Set physical constraints (faults, veins, intrusions) Generate 3D geological model Validate against geophysics
RMSP — GeologicAI (uncertainty module)
Geostatistical physical constraints for mineralogy estimation from well logs
PINN-adjacent; pure PINN tools 12–24 months away
UI Flow
Input geophysical well logs Apply petrophysical constraint layer Estimate mineralogy per interval Surface uncertainty per block Flag physics violations for review
Gap: no pure PINN commercial tool for hard rock mining
🕸️
Graph neural networks (GNNs)

GNNs model spatial relationships between geochemical, geophysical, and geological signals across large terranes — ideal for multi-signal target generation. Currently proprietary-only: the biggest white space in the market.

KoBold Metals (proprietary platform)
GNN-based multi-signal exploration targeting across large terranes
Proprietary — not commercially licensed
Inferred UI Flow
Ingest geochemical + geophysical layers Build spatial graph across terrane Run GNN to score spatial relationships Rank anomaly clusters Generate drill target recommendations
SensOre
AI-driven mineral targeting using spatial data relationships
Proprietary — exploration service, not a licensed tool
Inferred UI Flow
Submit regional datasets SensOre builds spatial signal graph GNN scores anomaly relationships Receive ranked target report
Biggest gap: no standalone commercial GNN geological modelling tool
🤖
Semi-supervised & transfer learning

The most widely deployed AI category in geoscience today. Learns geological patterns from small labelled datasets by leveraging large unlabelled data — enabling fast model bootstrapping and cross-deposit transfer.

Seequent Driver
ML-powered geological structure understanding with geologist override
Commercially deployed
UI Flow
Connect deposit dataset Driver learns geological structure (no coding) Auto-informs 3D model Geologist reviews + adjusts interpretations Export improved downstream model
Datarock
Geologist-in-the-loop core logging with IMDEX + ELEMISSION integrations
Commercially deployed
UI Flow
Core tray enters scanner (IMDEX / ELEMISSION) AI classifies lithology + alteration per interval Geologist validates flagged intervals Repeatable standardised log generated Push to database / resource model
🔍
Explainable AI (XAI)

AI outputs accompanied by traceable reasoning — critical for JORC / NI 43-101 compliance, where resource estimates must be defensible. Currently only niche vendors implement this; major platforms (Surpac, Vulcan) do not.

Datarock
Production-ready explainable AI for core characterisation — JORC/NI 43-101 ready
Commercially deployed
UI Flow
AI classifies core interval Confidence score surfaced per classification Feature attribution shown (which spectral bands drove result) Geologist approves / overrides with audit trail Compliant log exported for JORC report
RMSP — GeologicAI
Spatial uncertainty per block as a form of explainability for resource estimates
Commercially deployed
UI Flow
Run block model estimation Uncertainty value generated per block Colour-coded uncertainty heatmap in 3D viewer Filter high-uncertainty zones for follow-up drilling
Gap: XAI not yet in Surpac, Vulcan, or other major platforms
🛰️
Multimodal AI

Simultaneous fusion of multiple sensor streams (hyperspectral, XRF, LiDAR, RGB, magnetic susceptibility, LIBS) at the pixel level. GeologicAI is the most complete implementation in the market.

GeologicAI (RMSP + Lumo Analytics)
5-sensor mobile scanning unit — the only fully integrated critical minerals sensor platform
Commercially deployed
UI Flow — Scanning
Drive mobile trailer to drill site Auto-scan core trays (SWIR/VNIR + XRF + RGB + LiDAR + mag. susc.) AI stitches 5 sensor streams at pixel level LIBS adds rare-earth + light element detection (Lumo) Integrated mineralogy + structure model output
UI Flow — Analysis
Open fused dataset in RMSP Query mineralogy by interval or element Cross-reference with geophysics / block model Generate drill targets or resource update
Gap: no open API standard across sensor types
Datarock
RGB + hyperspectral core imaging with AI classification
Commercially deployed
UI Flow
Scan core with RGB + SWIR camera AI fuses visual + spectral signals Classify lithology + alteration + mineralogy Export structured log to IMDEX / database

# FMF26 Mobile App — Site Map & Feature Specification

## Overview

The FMF26 app is the official digital companion for the Future Minerals Forum 2026, held 13–15 January 2026 at the King Abdulaziz International Conference Centre in Riyadh, Saudi Arabia. It serves as a dedicated platform for the global minerals community to access event content, discover innovations across the metals value chain, and connect with delegates, sponsors, and speakers before and during the event.

## Core App Features

### 1. Pre-Event Engagement
| Feature | Description |
|---------|-------------|
| **Industry Engagement** | Connect with the minerals community ahead of FMF 2026 to discover latest innovations and technologies across the global metals value chain. |
| **Delegate Connect Sync** | Seamless integration with the FMF Delegate Connect platform for early networking and meeting scheduling. |
| **Profile Activation** | Set up professional profile with sector, role, commodities of interest, and region for intelligent matching. |

### 2. Search & Discovery
| Feature | Description |
|---------|-------------|
| **Speaker Search** | Find and filter global leaders, policymakers, mining CEOs, tech leaders, and government ministers. |
| **Product & Service Search** | Discover innovations and technologies across the metals value chain. |
| **Exhibitor Discovery** | Find exhibitors relevant to your business with intelligent filtering by product, service, and country. |
| **AI-Powered Matching** | FMF Global Connect uses intelligent filters and concierge support to recommend high-value meeting partners. |

### 3. Networking & Meetings
| Feature | Description |
|---------|-------------|
| **Intelligent Matching** | AI-powered algorithm recommends meeting partners based on sector, role, commodities, and region. |
| **Meeting Scheduling** | Pre-schedule and manage one-to-one meetings before and during the event. |
| **In-App Chat** | Direct messaging with delegates, file sharing, and helpdesk contact. |
| **Connection Management** | Send, accept, and manage connection requests. |

### 4. Event Navigation
| Feature | Description |
|---------|-------------|
| **Real-Time Information** | Live updates on sessions, locations, and schedule changes. |
| **Precise Location Services** | GPS-level navigation for venue layout, entrances, facilities, and transportation points. |
| **Digital Badge & QR Code** | On-device badge generation for seamless check-in and access control. |
| **Activity Logging** | Scan-based entry tracking at event checkpoints. |

### 5. Content & Media
| Feature | Description |
|---------|-------------|
| **FMF TV Integration** | Access to the Forum's exclusive video channel broadcasting senior industry voices. |
| **Live & On-Demand Content** | Stream sessions and watch replays. |
| **Programme Access** | Full agenda with nine strategic themes covering critical minerals, energy transition, and sustainable value chains. |

### 6. Communication Channels
| Feature | Description |
|---------|-------------|
| **Multi-Channel Alerts** | Push notifications, email, SMS, and WhatsApp messaging for updates and logistics. |
| **Transportation Integration** | Taxi/Uber integrations for venue transfers. |
| **Logistics Management** | Hotel, transport, and access-level permission coordination. |

## Technical & Privacy Specifications

| Specification | Detail |
|---------------|--------|
| **Data Encryption** | Encryption of data in transit |
| **Security** | Secure hosting, strict access controls, continuous monitoring, periodic audits |
| **Location Data** | GPS-level precise location (optional, can be disabled); not stored permanently |
| **Chat Encryption** | Encrypted depending on system configuration |
| **Data Retention** | Data retained only as long as necessary for event delivery and legal requirements; securely deleted or anonymized after |
| **Minimum Age** | Not intended for children under 16 |
| **Platforms** | Desktop (Delegate Connect web) + Mobile (iOS & Android apps) |

## Key Event Context (FMF 2026)

The 5th edition of the Future Minerals Forum brings together:
- **100+** government representatives at the Ministerial Roundtable
- **70+** international organizations, industry associations, NGOs
- Top-tier mining company CEOs, investors, and industry pioneers
- **Nine strategic themes** focused on critical minerals, energy transition, innovation, and sustainable value chains
- **Regional focus**: Africa, Central America, Latin America, and Central Asia supplier nations

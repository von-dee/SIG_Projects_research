---

## 1. App Overview

The FMF26 mobile app is the official companion application for the Future Minerals Forum — the world's premier government-led minerals platform. It serves attendees, exhibitors, speakers, and industry stakeholders before, during, and after the event by providing seamless access to event content, networking tools, and exhibitor discovery.

## 5. Key User Journeys

### Attendee Pre-Event Journey
1. Download app → Register / Log in → Receive digital badge
2. Browse programme → Bookmark sessions → Export to calendar
3. Search exhibitors relevant to business → Save exhibitors
4. Browse attendee directory → Send connection requests → Schedule meetings

### On-site Attendee Journey
1. Open app → Show QR badge at registration desk
2. View Today's agenda → Navigate to session room via floor map
3. Scan exhibitor/attendee badges → Chat or schedule follow-up meeting
4. Receive live push notifications for programme changes

### Exhibitor Journey
1. Log in → Access exhibitor profile → Verify product/service listings
2. View inbound connection requests from attendees
3. Manage meeting requests → Confirm meeting times and locations
4. Monitor and respond to attendee messages

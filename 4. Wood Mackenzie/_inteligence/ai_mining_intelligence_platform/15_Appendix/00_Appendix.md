# Appendix

## A. Glossary of Terms

| Term | Definition |
|------|------------|
| **AISC** | All-In Sustaining Cost — total cost of production including sustaining capital |
| **C1 Cost** | Cash operating cost per unit of production (direct costs only) |
| **EV** | Enterprise Value — market cap + debt - cash |
| **LCE** | Lithium Carbonate Equivalent — standard unit for lithium production |
| **NAV** | Net Asset Value — present value of future cash flows from reserves |
| **NRR** | Net Revenue Retention — revenue from existing customers including expansions and churn |
| **RAG** | Retrieval-Augmented Generation — AI technique combining retrieval with generation |
| **TCFD** | Task Force on Climate-related Financial Disclosures |
| **AIS** | Automatic Identification System — vessel tracking technology |
| **JORC** | Joint Ore Reserves Committee — mineral resource reporting standard |
| **NI 43-101** | Canadian mineral resource reporting standard |
| **PEA** | Preliminary Economic Assessment — early-stage project evaluation |
| **PFS** | Pre-Feasibility Study — intermediate project evaluation |
| **DFS** | Definitive Feasibility Study — final project evaluation |
| **Offtake** | Agreement to purchase future production |
| **Spot Price** | Current market price for immediate delivery |
| **Forward Curve** | Future prices across different delivery dates |
| **Contango** | Forward price > spot price |
| **Backwardation** | Forward price < spot price |

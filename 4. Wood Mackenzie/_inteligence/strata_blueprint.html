03 - Architecture

Seven-layer stack from raw data to intelligence

Cloud-native, horizontally scalable, and independently deployable. Each layer is a microservice designed to grow with data volume and user demand.

L1
Data Ingestion - Agentic Crawlers
Scheduled and event-triggered Python agents collecting from EDGAR, SEDAR, ASX, LME, CME, satellite imagery, and shipping data. Kafka event bus decouples ingestion from processing.
PythonPlaywrightKafkaSentinel-2
L2
Data Normalisation & Enrichment
LLM-powered extraction pipelines pull structured cost components from unstructured filings. Knowledge graph maps relationships between companies, assets, and commodities.
Claude APIdbtSparkNeo4j
L3
Multi-Model Storage Layer
Time-series in TimescaleDB, structured data in PostgreSQL with row-level security, documents in Pinecone vector store, geospatial in PostGIS, raw files in S3.
PostgreSQLTimescaleDBPineconeNeo4j
L4
AI Orchestration - Agents + RAG
Multi-agent system on LangGraph with specialised sub-agents for forecasting, extraction, synthesis, and alerts. Supervisor agent routes queries and assembles cited responses.
Claude SonnetLangGraphpgvectorLangSmith
L5
Quantitative Models Layer
Parameterised supply/demand models per commodity. Monte Carlo simulation for price scenarios. All assumptions transparent and user-overridable via the frontend.
PythonNumPySciPyFastAPI
L6
API & Backend Services
FastAPI server with REST and GraphQL endpoints. Celery workers for background jobs. Auth0 with SAML/SSO. Multi-tenancy enforced at the database layer.
FastAPIGraphQLCeleryAuth0
L7
Frontend - Dashboard + Conversational UI
React SPA with interactive charts, cost curves, real-time price feeds, chat-over-data interface with inline citations, and alerts management console. White-label available.
ReactTypeScriptD3.jsTailwind

07 - Risk Factors & Mitigations

What could go wrong, and how we handle it

Every risk has a documented mitigation. The highest-severity items have multiple layers of protection and early warning systems.

AI Hallucination in Cost Data
High
Every factual claim in AI outputs is linked to a source document or database record. Red-teaming during alpha. Commodity analyst QA layer on extraction pipeline output.
Data Source Changes / Scraping Breaks
Medium
Redundant data sources per commodity. Automated monitoring on ingestion pipeline health. Fallback to cached data with staleness indicators.
Wood Mackenzie Launches Chat Interface
Medium
Their moat is analyst depth, not tech. A chat layer on static data does not give real-time or private data integration. Our differentiation is architectural, not UX.
LLM API Cost at Scale
Medium
Cost-per-query optimisation: cache common queries, use smaller models for classification tasks, batch extraction jobs. Target <$0.10/query average at scale.
Data Licensing / IP Disputes
Low-Med
All sources are public filings and free government data. Legal review of data collection practices in Year 1. Avoid scraping behind paywalls.
Enterprise Sales Cycle Length
Medium
Mitigate with bottom-up PLG: individual users adopt Explorer/Pro, pull enterprise procurement. Reduces cold outbound dependency.
Key Person Dependency
High
Hire commodity analyst by Month 5 to distribute domain knowledge. Document all source validation logic explicitly.
Commodity Price Data Accuracy
High
Multiple source cross-validation for all price series. Automated anomaly detection on time series. Manual review alert when deviation > 2 standard deviations.

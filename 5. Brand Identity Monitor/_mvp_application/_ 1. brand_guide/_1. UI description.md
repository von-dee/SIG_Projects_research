**UI Description — DentalPro Project Dashboard**

This is a project management dashboard (similar to Notion, Linear, or Asana) showing a workspace for a dental brand called DentalPro. The interface is divided into three main panels:

**Left sidebar** — A vertical navigation panel listing workspaces grouped under platforms (Dribbble, Behance). Each workspace has a colored dot indicator and sub-items. The active item is DentalPro, expanded to show four sub-tasks: Mobile app, Website, Brand identity, and Logo design.

**Center panel** — The main content area showing the DentalPro project. At the top is a project header with the logo, creation date (Dec 3), due date (Jan 24), and tracked time. Below that are five navigation tabs (Overview, List, Dashboard, Timeline, Calendar) with Dashboard currently active. Three filter chips appear below: Deadline, Department: Web development, and Priority. The main view is a kanban board with three columns — Ice box (3 cards), In Progress (8 cards), and Discussion (3 cards) — each containing task cards with descriptions, assignee avatars, tags (Docs, Prepare, Illustrations, Prototype, Coding), and comment/attachment counts.

**Right sidebar** — Shows Recent Project Work (two cards with progress percentages: 71% and 43%), a Time Management section with a line chart showing hours tracked across the week (currently 36h 10m total), and a Team Chat panel with a live conversation thread.

The overall design aesthetic is clean, minimal, and modern — a white/light gray base with purple as the primary accent color.

---The brand guide covers everything extracted from the UI:

**Colors** — Deep purple (#7B68EE) is the clear primary, used for all active states, CTAs, and progress numbers. The lavender tint (#EDE9FF) cushions it on light surfaces. Amber, green, and a neutral charcoal round out the functional palette.

**Typography** — No custom typeface is declared in the UI; it uses a clean system sans-serif. The scale is simple: 22px display → 16px section headers → 14px body → 12px meta. Weight stays at 400/500 only — never bold-heavy.

**Tags** — Each tag category has its own color family: purple for Docs/Prepare, amber for Illustrations, green for Coding, and gray for Prototype. All use pill shapes with full border-radius.

**Components** — Tabs use a filled purple pill for the active state. Cards are white with hairline borders and 12px radius. Filter chips are borderlined with chevrons.

**Spacing** — Built on a 4px grid, 16px gutters, 12px card radius, 20px full-radius for pills.

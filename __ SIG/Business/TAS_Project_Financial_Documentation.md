## PROJECT OVERVIEW
**Project Name:** Traceability & Assurance System (TAS) for Ghana Gold Board  
**Contract Duration:** 18 months from contract signature  
**Implementation Phase:** 6 months (Design, Build, Integration, Pilot, Go-Live)  
**Stabilisation Phase:** 12 months (Support, Optimisation, Handover)  
**Procurement Method:** Quality and Cost Based Selection (QCBS)  
**Regulatory Framework:** Public Procurement Act, 2003 (Act 663 as amended)

## 7. KEY PERFORMANCE INDICATORS (KPIs) - FINANCIAL

### 7.1 Budget Performance KPIs

- **Budget Variance** = (Actual Cost - Budgeted Cost) / Budgeted Cost × 100%
  - Target: ≤ ±10%
  
- **Cost Performance Index (CPI)** = Earned Value / Actual Cost
  - Target: ≥ 0.95
  
- **Budget Consumption Rate** = Cumulative Actual Cost / Total Budget × 100%
  - Should align with % project completion
  
- **Forecast Accuracy** = |Forecast Cost - Actual Cost| / Actual Cost × 100%
  - Target: ≤ 5%

### 7.2 Financial Efficiency KPIs

- **Cost per User Onboarded** = Total Onboarding Cost / Number of Users
  
- **Cost per Transaction Processed** = Total Operational Cost / Number of Transactions
  
- **Infrastructure Cost Efficiency** = Infrastructure Cost / Number of Users Supported
  
- **Training Cost Efficiency** = Training Cost / Number of Trainees

### 7.3 Payment Performance KPIs

- **Payment Milestone Achievement Rate** = Milestones Achieved on Time / Total Milestones × 100%
  - Target: ≥ 90%
  
- **Invoice Payment Cycle Time** = Average days from invoice to payment
  - Target: ≤ 30 days
  
- **Vendor Payment Timeliness** = Payments Made on Time / Total Payments × 100%
  - Target: ≥ 95%

### 7.4 Value for Money KPIs

- **Return on Investment (ROI)** = (Benefits - Costs) / Costs × 100%
  
- **Cost Avoidance** = Identified savings through value engineering
  
- **Procurement Savings** = (Initial Quote - Final Price) / Initial Quote × 100%

### 7.5 Compliance KPIs

- **Procurement Compliance Rate** = Compliant Procurements / Total Procurements × 100%
  - Target: 100%
  
- **Tax Compliance Rate** = Tax Returns Filed on Time / Total Tax Returns × 100%
  - Target: 100%
  
- **Audit Finding Closure Rate** = Findings Resolved / Total Findings × 100%
  - Target: ≥ 90% within agreed timeframe
